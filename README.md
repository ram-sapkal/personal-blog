# personal-blog
Personal Blog Platform is a full-stack web application built with the MERN stack (MongoDB, Express, Next.js, Node.js)

# Personal Blog Platform (MERN Stack)

## Features

- User Sign Up / Login
- JWT Auth
- Create/View Posts
- Filter by Author
- Protected Dashboard

## Tech Stack

- MongoDB + Mongoose
- Express + Node.js
- Next.js 14(Frontend)
- Tailwind CSS

---

## Backend Setup

```bash
cd backend
npm install
npm run dev
```
---

## Frontend Setup

```bash 
cd client
npm install
npm run dev
